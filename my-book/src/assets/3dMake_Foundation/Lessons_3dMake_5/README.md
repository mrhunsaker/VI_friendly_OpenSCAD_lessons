# Lesson 5: Safety Protocols and the Physical Fabrication Interface

**Estimated time:** 60-75 minutes

## Folder Contents

This folder contains supporting materials for Lesson 5:

- **Safety Checklists:** Pre-print, operation, and post-print verification templates
- **Reference Materials:** Material properties, printer documentation, maintenance schedules
- **Templates:** Print logs, incident reports, and maintenance records

## Learning Objectives

- Understand FDM printer operation and safety
- Follow fabrication protocols for safe printing
- Handle and inspect printed parts
- Maintain printer equipment
- Select appropriate materials for different applications
