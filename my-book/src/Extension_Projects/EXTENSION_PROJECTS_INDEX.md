# Extension Projects — Locations & Didactic Purposes

This file lists the extension projects, where they live, their primary didactic purpose, and the starter OpenSCAD file included in each folder.

-- **Your First Print**: [Your_First_Print/your-first-print.md](Your_First_Print/your-first-print.md)
  - Purpose: low-friction introduction to the printing workflow and slicer checks; suitable for first-time printers.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Your_First_Print/starter.scad) (basic cube example to slice and print).

-- **Real-Life Problem Solver**: [Your_Second_Print/your-second-print.md](Your_Second_Print/your-second-print.md)
  - Purpose: adapt existing models to meet user needs; practice iteration and changelogs.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Your_Second_Print/starter.scad) (small mounting/bracket scaffold).

-- **Dice Design Workshop**: [Dice_Dice_Dice/Dice_Dice_Dice.md](Dice_Dice_Dice/Dice_Dice_Dice.md)
  - Purpose: parametric design and mass/distribution testing; introduces testing methodology.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Dice_Dice_Dice/starter.scad) (parametric die scaffold).

-- **Bonus Print**: [Bonus_Print/bonus-print.md](Bonus_Print/bonus-print.md)
  - Purpose: scaling and simple modifications; multi-part printing and documentation.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Bonus_Print/starter.scad) (scaling example).

-- **Parametric Keychain**: [Parametric_Keychain/Parametric_Keychain.md](Parametric_Keychain/Parametric_Keychain.md)
  - Purpose: string together basic 2D → 3D workflows and introduce text/embossing concepts.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Parametric_Keychain/starter.scad).

-- **Snap-Fit Clip**: [Snap_Fit_Clip/Snap_Fit_Clip.md](Snap_Fit_Clip/Snap_Fit_Clip.md)
  - Purpose: teach tolerance testing and mechanical considerations for snap-fit joints.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Snap_Fit_Clip/starter.scad).

-- **Accessibility Audit**: [Accessibility_Audit/Accessibility_Audit.md](Accessibility_Audit/Accessibility_Audit.md)
  - Purpose: practical exercise in improving lessons for non-visual users; includes small annotated example.
  - Starter SCAD: [starter.scad](../assets/Extension_Projects/Accessibility_Audit/starter.scad) (annotated minimal example).
