## Practical Skills Assessment

### Navigation & Directory Operations

1. Navigate to your home directory and verify location
2. Create a folder structure: `Project\Models\Designs`
3. Navigate through all levels using different methods
4. Navigate back home using an absolute path
5. List all files and folders at each level

### File Operations

1. Create 5 text files with different names
2. Copy them to a backup folder
3. Rename one file
4. Move one file to a different folder
5. Delete one file safely
6. Verify all operations succeeded

### Output Redirection

1. Save a directory listing to a file
2. Append additional information to the file
3. Open and read the file in Notepad
4. Redirect help output to a file
5. Verify all saved files are correct

### Batch File Creation

1. Create a batch file that creates a project structure
2. Add comments explaining what it does
3. Use at least two variables
4. Test the batch file works correctly
5. Modify it and test changes

## Quiz - Comprehensive Command Prompt

### Navigation (5 questions)

1. How do you navigate to your home directory from anywhere?
2. What path would you use to go to Documents?
3. Explain the difference between `..` and `C:\`
4. How does Tab completion work?
5. If you're at `C:\Users\Name\Documents\Deep\Folder`, how do you go back to `C:\Users\Name`?

### File Operations (5 questions)

6. Write the command to create a folder called "archive"
7. Write the command to copy all `.txt` files to a backup folder
8. Write the command to safely delete one specific file
9. What wildcard would match files named file1, file2, file3?
10. How do you rename a file from `old.txt` to `new.txt`?

### Redirection (5 questions)

11. Write a command to save a directory listing to a file
12. What's the difference between `>` and `>>`?
13. How would you append the current path to an existing file?
14. Write a command to save help output to a file
15. Why is saving output to files important for screen readers?

### Batch Files (5 questions)

16. What does `@echo off` do?
17. Write a batch file line that creates a backup folder
18. How do you use variables in batch files?
19. What does `REM` do in a batch file?
20. Write a complete batch file that creates a project structure and lists it

## Practical Challenges

### Challenge 1: Project Setup (Intermediate)

Create a batch file that:
- Creates a new folder with the name you provide
- Inside it, creates Models, Prints, and Documentation folders
- Saves a listing to a log file
- Shows completion message

**Test it** by running and verifying the structure.

### Challenge 2: File Organization (Intermediate)

Create a batch file that:
- Finds all `.txt` files in Documents
- Copies them to a new "TextFiles" folder
- Saves a list of copied files
- Shows the results

**Test it** with actual files.

### Challenge 3: Backup System (Advanced)

Create a batch file that:
- Creates a backup folder with today's date in the name
- Copies all `.scad` files to the backup
- Saves a detailed log of what was copied
- Shows success/error messages

**Test thoroughly** before using with real files.
